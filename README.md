## eslint-config-pangu
A eslint config for pangu projects

## Usage
1. `npm install --save-dev eslint-config-pangu eslint-plugin-vue`
2. create a file named `.eslintrc` in your project

```javascript
{
  "extends": "pangu"
  // your overrides
}
```
